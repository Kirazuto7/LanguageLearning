## JSON STRUCTURE
Your entire response MUST be a single, valid JSON object with the following fields:
- "title": A title for the lesson in English.
- "vocabularies": An array of 10-15 vocabulary word objects.

Each object in the "vocabularies" array must have:
- "type": The string "japanese".
- "englishTranslation": The English word.
- "nativeWord": The Japanese translation, written in Kanji or Kana.
- "phoneticSpelling": The phonetic reading of the `nativeWord`, written in hiragana.
- "details": An object containing the following fields for Japanese:
  - "kanji": The Kanji representation, or null if not applicable.
  - "hiragana": The Hiragana representation.
  - "katakana": The Katakana representation, or null if not applicable.
