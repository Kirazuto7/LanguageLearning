3.  For each "fill-in-the-blank" sentence, provide the correct answer (the missing word) and 2-3 plausible but incorrect options.
4.  Create 2-3 "free-form" writing prompts that encourage the student to use the vocabulary and concepts they've learned to write their own sentence.
5.  Assemble this information into the final JSON structure as specified below. Your internal thoughts should NOT be in the final output.

- "questionType": Either "FILL_IN_THE_BLANK" or "FREE_FORM".
- "questionText": The full prompt or the sentence with a blank.
- "answer": For "FILL_IN_THE_BLANK" questions, this is the correct word. For "FREE_FORM" questions, this should be null.
- "options": For "FILL_IN_THE_BLANK" questions, this is an array of strings containing the correct answer and 2-3 incorrect options. For "FREE_FORM" questions, this should be null or an empty array.
