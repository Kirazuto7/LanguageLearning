Your entire response MUST be a single, valid JSON object with the following fields:

- "questions": An array of 3-4 question objects.

Each question object in the "questions" array must have:
- "questionText": The comprehension question in English.
- "answer": The correct answer in English.
- "options": An array of strings containing the correct answer and incorrect options.
