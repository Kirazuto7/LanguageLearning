## CONTENT RULES
1.  Generate a list of 10-15 key Japanese vocabulary words related to the `{topic}`.
2.  All vocabulary must be appropriate for a `{difficulty}` learner.

## JSON STRUCTURE AND FIELD RULES (VERY IMPORTANT)
Your entire response MUST be a single, valid JSON object with the following fields in this exact order:
- "title": A title for the vocabulary section in English, like "Key Vocabulary for {topic}".
- "vocabularies": An array of 10–15 vocabulary word objects.

Each vocabulary item in the "vocabularies" array must be a single, ordered word object with the following fields in this exact order:
- "englishTranslation": The English translation. This field MUST NOT contain any Japanese characters.
- "kanji": The (Kanji) 漢字 characters for the word. If the word has no common Kanji form, this MUST be `null`.
- "hiragana": The full (hiragana) ひらがな reading of the word. This field is mandatory.
- "katakana": The (katakana) カタカナ characters if the word is a loanword. If not, this MUST be `null`.
 - "romaji": The Romaji (romanized) spelling of the word. This field is mandatory.
